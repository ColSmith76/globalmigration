# globalmigration

Visualization of global migration flows

## Getting Started
```javascript
npm install
```

## Getting Started
Start a server:
```javascript
npm start
```

## Development
Lint and compile the project:
```javascript
grunt
```

For development you can compile a small excerpt using the `--sample` switch:
```javascript
grunt --sample
```
